Machine Learning — 4 Learning Paradigms
Categorized by level of human supervision required during training
Labeled data → Supervised No labels → Unsupervised Mixed labels → Semi-supervised Trial & error → Reinforcement
S
Supervised Learning
Input + correct output labels provided
Sub-types
Regression Classification
Regression: predicts a continuous number — salary, house price, temperature
Classification: predicts a category — spam/not spam, placed/not placed
Algorithm learns a mapping: inputs → correct outputs
REAL EXAMPLES
Predicting salary · Classifying emails · Diagnosing diseases · Housing price models
U
Unsupervised Learning
No labels — model finds hidden structure
Techniques
Clustering Dim. Reduction Anomaly Detection Association Rules
Clustering: group similar data — customer segments
Dim. Reduction: compress data while keeping key info
Anomaly Detection: find outliers — fraud, defects
Association Rules: diapers buyers also buy beer
REAL EXAMPLES
Customer segmentation · Fraud detection · Topic modeling · Market basket analysis
SS
Semi-Supervised Learning
Hybrid — small labeled + large unlabeled data
Why it exists
Labeling is expensive Hybrid approach
Labeling data is costly and time-consuming in practice
Small labeled set teaches the model basic structure
Large unlabeled pool extends the learning efficiently
Much more efficient than fully labeling all data
REAL EXAMPLES
Medical image labeling · Web content classification · Speech recognition training
RL
Reinforcement Learning
Agent learns via rewards & punishments
Core Loop
Agent Environment Reward signal Policy optimization
Agent takes actions in an environment
Gets rewards for good actions, punishments for bad
Optimizes its policy to maximize cumulative reward
Does NOT rely on a static labeled dataset
FAMOUS EXAMPLE
AlphaGo — learned to master the board game Go by playing millions of games against itself
Supervised — needs labels for every sample
Unsupervised — zero labels, discovers structure
Semi-supervised — few labels + many unlabeled
Reinforcement — learns by interacting with environment
