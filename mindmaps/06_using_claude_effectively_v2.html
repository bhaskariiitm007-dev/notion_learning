Iterative Prompting
Refine output in layers — don't restart
1
Give full context upfront for the strongest first response
2
Read the output, then ask for one specific change at a time
3
Use select + reply to reference a specific block of the last response
4
Ask Claude to add approximate costs, change tone, or make it more adventurous — incremental refinement
🔁 Layer by Layer 📌 Select + Reply 🎯 One Change at a Time
