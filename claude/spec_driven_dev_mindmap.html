Common Pitfalls to Avoid
What breaks SDD and sends you back to vibe-land
!
Skipping the spec — no acceptance criteria means no objective way to define done
!
Vague requirements — ambiguity gives AI too much freedom to fill gaps incorrectly
!
Ignoring edge cases — production systems fail at boundaries the spec never anticipated
!
Coding on main branch — spec-driven changes can be wide-reaching; always isolate on a feature branch
!
Auto-approving AI output — skipping human review gate collapses back into vibe coding
🚫 No Spec🌫️ Vague Reqs🛡️ Missing Edges🤖 Blind Approval
